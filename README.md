# Flight-Booking-System
A desktop-based flight booking system built using Java Swing, featuring flight search, booking, cancellation, billing, and zone visualisation. Designed with modular architecture and a clean UI for a smooth user experience.

🧩 Features
- 🔍 Search Flights by origin and destination
- 🧑 Passenger Information entry and validation
- 📄 Bill Generation after successful booking
- ❌ Cancel Bookings and view cancellation history
- 🌍 Flight Zone Viewer (e.g., North, South, East, West)
- 🔐 Logout Functionality for session control

🛠️ Tech Stack
  Layer         |     Technology
----------------|----------------        
  Language      |     Java  
  UI Framework  |     Swing  
  Architecture  |     MVC (Modular)  
  
📦 Modules
- model/ → Flight, Passenger, Booking classes
- service/ → FlightService, BookingService
- ui/ → Panels for booking, cancellation, billing, zone view, and logout
- Main.java → Entry point with JFrame setup
